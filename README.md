# 🏥 Hospital Patient Survey Analysis Dashboard (Power BI)

This project is an interactive **Power BI dashboard** developed to analyze hospital patient survey data and provide insights into patient satisfaction, service quality, and hospital performance.

## 📌 Project Overview

The dashboard transforms raw hospital survey data into meaningful visualizations that help identify trends, measure patient experience, and support data-driven decision-making.

## 🚀 Key Features

* Interactive dashboards with slicers and filters
* Patient satisfaction analysis
* Department-wise performance comparison
* Survey response summaries
* Dynamic tables and pivot reports
* Clean and user-friendly dashboard design

## 🛠️ Tools & Technologies

* **Power BI Desktop**
* Power Query
* DAX (Data Analysis Expressions)
* Data Modeling
* Interactive Visualizations

## 📊 Dashboard Highlights

* Overall survey performance overview
* Department-wise insights
* Filter by survey categories
* Interactive reports for better exploration
* Data-driven KPIs and summary tables

## 🎯 Objectives

* Analyze hospital patient feedback.
* Identify areas requiring service improvement.
* Enable stakeholders to make informed decisions through visual analytics.
